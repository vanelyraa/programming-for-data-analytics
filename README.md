# Programming for Data Analytics Repository

## Project Purpose  
This repository is organized to contain both my assignments and final project submissions for the Programming for Data Analytics module completed as part of the Higher Diploma in Science in Computing in Data Analytics at ATU.
The goal of this project is to demonstrate my skills in: Python programming, data retrieval from various sources (URLs, CSV files, JSON), data cleaning and analysis, data manipulation with pandas, statistics calculations, data visualisation using matplotlib and seaborn.

## Folders
- **assignments**: This folder includes: setup instructions, dependencies, and implementations for all lecture assignments. Each assignment is in a separate Python file or Jupyter Notebook with a dedicated README file.
